Communication par l’intermédiaire du Supplier Portal

Le Supplier Portal peut prendre un certain temps pour s'y habituer, mais à long terme, il sera plus facile pour tout le monde d'avoir toutes les communications concernant une réservation à portée de main. L'objectif principal est de limiter les divers e-mails «en vrac» au moyen de ce portail fournisseur et de garder toutes les communications clairement organisées. Comment cela fonctionne-t-il?

Dès qu’un nouveau rapport ou une nouvelle « rooming list » est disponible, où dès qu’une nouvelle réservation, une modification ou une annulation est effectuée, vous recevez automatiquement une notification sur la page « ANWB Messages ». Ce système est à sens unique, ce qui signifie que vous pouvez uniquement recevoir des messages (mais pas y répondre).

Pour chaque numéro de réservation, vous pouvez toutefois communiquer avec notre service « Operations » par le biais des mémos. Vous pouvez voir la date et l’heure de chaque mémo et le nom de l’employé(e) qui l’a rédigé. Lorsque nous vous envoyons un nouveau mémo, vous recevez une notification sur la page « ANWB Messages ». Lorsque vous répondez à un message ou que vous en ajoutez un nouveau, notre service « Operations » en est automatiquement informé et vous répondra si nécessaire.

Il y a plusieurs choses que vous pouvez faire pour communiquer par le biais du Supplier Portal :

A. Ajouter un nouveau mémo

Si vous souhaitez nous envoyer un message contenant des informations ou des questions, vous pouvez cliquer sur le bouton « Add a new memo ». Un champ de texte s’ouvre dans lequel vous pouvez écrire un message de 4000 caractères maximum (si vous dépassez ce nombre, vous pouvez diviser le message en deux et envoyer un second mémo). Cliquez sur « OK » pour nous envoyer le message.

B. Répondre à un mémo

Si vous avez reçu un message de notre part auquel vous souhaitez ou devez répondre, vous pouvez cliquer sur le bouton « Reply to ANWB Reizen ». Un champ de texte s’ouvre alors, dans lequel vous pouvez écrire votre réponse. Cliquez sur « OK » pour nous envoyer le message.

C. Marquer les messages comme étant terminés

Une fois que vous avez répondu à un message de notre part ou que vous avez simplement lu les informations sans avoir besoin de répondre, vous pouvez marquer le message comme étant terminé en cliquant sur le bouton « Mark completed ». Il est ainsi facile pour vous et vos collègues de savoir si l'un d'entre vous a traité l'information.

En outre, en cliquant sur ce bouton, la notification du message reçu sur la page "Messages ANWB" disparaît. Il est ainsi plus facile de garder une bonne vue d'ensemble sur cette page spécifique.

D. Ajouter une note interneD.

Si vous souhaitez noter certaines informations pour vous-même ou vos collègues sans nous envoyer de message, vous pouvez ajouter une note interne. Cette note ne sera visible que pour vous et vos collègues, pas pour nous. Cliquez sur le bouton « Add an Internal note » pour ouvrir un champ de texte. Une fois que vous avez saisi les informations, cliquez sur « OK » pour les enregistrer.
