Kommunikation über das Supplier Portal

Das Supplier Portal mag etwas gewöhnungsbedürftig sein, aber auf lange Sicht wird es für jeden einfacher sein, die gesamte Kommunikation über eine Reservierung an einem Ort gebündelt zu haben. Das Hauptziel ist es, die Anzahl der "losen" E-Mails zu begrenzen und die gesamte Kommunikation klar zu organisieren. Wie funktioniert das?

Wenn ein neues Buchungsdokument („Report“) oder eine neue Zimmerbelegungsliste („Roominglist“) verfügbar ist oder es neue, geänderte oder stornierte Buchungen gibt, erhalten Sie auf der Seite „ANWB Messages“ eine Meldung. Dieses System funktioniert nach dem Einbahnstraßenprinzip, wobei Sie nur Nachrichten erhalten können.

Im Rahmen einer individuellen Buchungsnummer können Sie jedoch über die Memos mit unserer Operations-Abteilung kommunizieren. Sie können das Datum und die Uhrzeit des Memos sowie den Namen des Mitarbeiters, der das Memo geschrieben hat, sehen. Wenn wir ein neues Memo geschickt haben, werden Sie darüber über die Seite „ANWB Messages“ informiert. Wenn Sie auf eine Nachricht reagieren oder eine neue Nachricht hinzufügen, wird unsere Operations-Abteilung automatisch informiert und erforderlichenfalls reagieren.

Es gibt innerhalb des Supplier Portals folgende Kommunikationsmöglichkeiten:

A. Hinzufügen einer neuen Nachricht

Wenn Sie uns eine Nachricht mit Informationen oder Fragen senden möchten, können Sie auf die Schaltfläche “Add a new memo“ klicken. Es öffnet sich ein Textfeld, in das Sie eine Nachricht von höchstens 4000 Zeichen schreiben können (wenn Sie diese Zeichenzahl überschreiten, können Sie die Nachricht in zwei Abschnitte aufteilen und eine zweite Kurzmitteilung senden). Indem Sie auf “OK“ klicken, wird uns die Nachricht zugeschickt.

B. Auf eine Nachricht reagieren

Wenn Sie von uns eine Nachricht erhalten haben, auf die Sie reagieren wollen oder müssen, können Sie auf die Schaltfläche “Reply to ANWB Reizen“ klicken. Es öffnet sich wieder ein Textfeld, in das Sie Ihre Antwort setzen können. Indem Sie auf “OK“ klicken, wird uns die Nachricht zugeschickt.

C. Nachrichten als erledigt markieren

Wenn Sie einmal auf eine Nachricht reagiert haben oder die Informationen gelesen haben, ohne darauf antworten zu müssen, können Sie die Nachricht als erledigt markieren, indem Sie auf die Schaltfläche “Mark completed“ klicken. So können Sie und Ihre Kollegen bequem sehen, wie mit den Informationen verfahren worden ist. Außerdem verschwindet nach dem Klicken auf diese Schaltfläche auf der Seite “ANWB Messages“ die Meldung, dass eine Nachricht eingegangen ist. Auf diese Art und Weise können Sie auf jener Seite besser den Überblick behalten.

D. Hinzufügen einer internen Notiz

Wenn Sie Informationen für sich selbst oder Ihre Kollegen notieren möchten, können Sie eine interne Notiz hinzufügen. Diese Notiz ist nur für Sie selbst und Ihre Kollegen und also nicht für uns sichtbar. Wenn Sie auf die Schaltfläche „Add an Internal note“ klicken, öffnet sich ein Textfeld. Nachdem Sie die Informationen eingegeben haben, klicken Sie zum Speichern auf „OK“.
