Reservations - Informationen zu einer Buchung

Das Supplier Portal enthält verschiedene Registerkarten, auf denen Reservierungen einfach angezeigt werden können. Weitere Informationen finden Sie auch in diesem Portal. Das Versenden einzelner E-Mails ist nicht mehr erforderlich. Im Portal finden Sie ganz einfach alles über eine Buchung.

Sie finden diese Informationen im Supplier Portal:

A. Informationen zu Teilnehmern/Mitgliedern
B. Leistungselemente / Informationen zu den gebuchten Artikeln
C. Essenzen und Präferenzen/Kundenwünsche
D. Zusätzliche Informationen zur Buchung
E. Angaben zur Referenznummer

A. Buchungen/Teilnehmer

Zu jeder Buchung werden alle Informationen, die wir zu den Teilnehmern haben, angezeigt. Aus Datenschutzgründen fordern wir bei unseren Kunden ein Mindestmaß an Daten an. Sollten Sie mehr Informationen als die bereitgestellten Daten benötigen, setzen Sie sich bitte mit uns in Verbindung. Wir schauen dann, ob wir die erbetenen Informationen bereitstellen können.

B. Leistungselemente / Informationen zu den gebuchten Artikeln

Ein Leistungselement ist eine gebuchte Leistung. Dabei kann es sich um eine Gruppen- oder individuelle Reise, ein Hotel, einen Transfer, eine Verlängerung usw. handeln. Bitte berücksichtigen Sie, dass Sie nur die Leistungselemente sehen können, die an Sie als Leistungserbringer gekoppelt sind. Wenn wir beispielsweise über Urlaubsreiseangebote verfügen, in denen wir Leistungselemente verschiedener Leistungserbringer kombinieren, können Sie nur die Leistungselemente sehen, die Sie bereitgestellt haben.

Dabei ist zudem zu berücksichtigen, dass nicht alle Leistungselemente von allen Teilnehmern gebucht werden. Bei einem Ausflug könnte es sich beispielsweise ergeben, dass nur einer der zwei Teilnehmer sich dafür entschieden hat, diesen zu buchen. Innerhalb des Supplier Portals können Sie sehen, welchen Teilnehmern die einzelnen Leistungselemente zugewiesen worden sind. Auch können Sie das Datum sehen, an dem die Leistungselemente gebucht worden sind.

In der Reservierung wird auch der Kaufpreis für das jeweilige Element angegeben.

C. Essenzen und Präferenzen/Kundenwünsche

Bei dem Posten Essenzen und Präferenzen handelt es sich um bestimmte Wünsche unserer Kunden. Manche dieser Punkte gelten möglicherweise für die gesamte Reisegesellschaft (zum Beispiel “Connecting rooms“, also benachbarte Zimmer), während andere nur für einzelne Personen zutreffen (beispielsweise „diabetic meals“ für Diabetiker). Innerhalb des Supplier Portals können Sie sehen, welchen Teilnehmern die einzelnen Wunschelemente zugewiesen wurden.

Eine Präferenz ist ein Wunsch des Kunden. Der Kunde ist sich bewusst, dass es sich um einen Wunsch handelt und dass daraus keine Rechte abgeleitet werden können. Wenn es eine Präferenz in der Reservierung gibt, braucht ANWB Reizen keine Rückmeldung dafür.

Bitte berücksichtigen Sie, dass es sich bei manchen Wünschen um lebensnotwendige Leistungselemente handeln kann, beispielsweise medizinische Essenzen. Wenn Sie solch einer Essenz nicht gerecht werden können, muss der betreffende Kunde die Buchung kostenlos stornieren können. Eine medizinische Essenz wird auch immer auf Anfrage sein, so dass die Reservierung nicht endgültig ist, bis diese Essenz bestätigt ist.

D. Zusatzinformationen

Die Zusatzinformationen können für die Buchung erforderliche Angaben enthalten. Bei einer Überfahrt mit einer Fähre könnten die Zusatzinformationen beispielsweise die Maße des Fahrzeugs betreffen. Oder im Falle beispielsweise eines Skiurlaubs können sich die Zusatzinformationen auf die Schuhgrößen der Teilnehmer beziehen, sodass passendes Skimaterial gebucht werden kann. Je nach den Informationen, die Sie als Leistungserbringer benötigen, um eine Buchung umsetzen zu können, können wir bestimmte Fragen an unsere Website koppeln, die an ein bestimmtes Urlaubsreiseangebot gekoppelt sind. Setzen Sie sich mit uns in Verbindung, wenn bestimmte Informationen, die Sie benötigen, fehlen.

E. Referenznummer / Bezugsnummer

Alle Buchungen haben eine individuelle Buchungsreferenznummer, die unser System automatisch generiert. Außerdem können Sie einer Buchung auch Ihre eigene Referenznummer hinzufügen. Über die Seite “Individuals“ können Sie nach dieser „Supplier reference“ suchen, was praktisch sein kann, wenn Sie Buchungen innerhalb des Systems ihrer eigenen Organisation auf eine andere Art und Weise verwalten.


Neu seit November 2021:

Excel-Export jetzt mit erweiterten Reservierungsdaten:

Es ist jetzt möglich, einen umfangreichen Export ausgewählter Reservierungen herunterzuladen. Als Anbieter können Sie auf der Seite „Individuell“ nach einer Liste von Reservierungen auf der Grundlage von Filtern suchen. Über den 'Export to Excel'-Button wird eine Excel-Liste mit den pro Reservierung erweiterten Reservierungsdaten erstellt:
- Teilnehmer
- Elemente und Leistungsarten (Zimmertyp, Leistung) + Zeitraum

So können Sie als Anbieter leicht überprüfen, ob alles ordnungsgemäß in Ihren eigenen Systemen verbucht wurde.

Auch werden Änderungen markiert und es gibt eine neue Schaltfläche für die Bestätigung von Änderungen.

Wenn eine Reservierung durch ANWB geändert wird, wird die Änderung nun in der Reservierungsübersicht durch Markierung des Feldes angezeigt. Wenn frühere Informationen geändert wurden, färbt sich der Hintergrund orange. Wenn vorherige Informationen gelöscht werden, wird der Hintergrund rot.
